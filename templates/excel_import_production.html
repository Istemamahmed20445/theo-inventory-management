Excel Format Guide
Required Columns:
  1. Column A: Product Name (must match existing products)
  2. Column B: Production Quantity
  3. Column C: Status (pending, in_progress, completed)
  4. Column D: Notes (optional)
Example Format:
A B C D
T-Shirt 100 pending Summer collection
Jeans 50 in_progress Blue denim
Note: Product names must exactly match existing products in your inventory.
Import Instructions
Before Importing:
  • Ensure all products exist in your inventory
  • Use exact product names as they appear in the system
  • Save your Excel file as .xlsx format
  • Remove any empty rows
After Importing:
  • Check the production orders page for imported data
  • Verify quantities and statuses are correct
  • Update any incorrect information manually
  • Start production workflow as needed
Important: This import will create new production orders. Make sure your data is accurate before uploading.
